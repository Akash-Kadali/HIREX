About HIREX

High Resume eXpert ‚Äî A job-aware AI resume optimizer purpose-built for LaTeX users. HIREX aligns your resume to any JD, preserves structure/macros, and offers optional tone refinement.

Project Overview

HIREX transforms your LaTeX resume into a job-specific, ATS-optimized document using advanced AI. It ensures factual precision, professional tone, and strong keyword alignment while preserving your original LaTeX structure.

System Architecture

  1. Frontend: Responsive HTML+JS using HIREX UI v1.2.0 (glass-neon design), theme persistence, and a modern Humanize switch.
  2. Backend: FastAPI engine for AI inference, LaTeX compilation, and secure file management.
  3. Base LaTeX: Single canonical file data/samples/base_resume.tex used for all runs (no frontend uploads).
  4. AI Layer: GPT-based optimizer + optional Humanize provider for tone calibration.

The architecture prioritizes accuracy, transparency, and privacy ‚Äî no unnecessary data leaves your environment.
